# ph-rice-price

[![npm version](https://img.shields.io/npm/v/ph-rice-price.svg)](https://www.npmjs.com/package/ph-rice-price)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Philippine rice price calculation and conversion utilities

## Installation

```bash
npm install ph-rice-price
```

## Functions

- `convertCavanToKg` — Convert cavan (sack) to kilograms
- `convertKgToCavan` — Convert kilograms to cavans
- `computeRetailPrice` — Compute retail price from wholesale per cavan
